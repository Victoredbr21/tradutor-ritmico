# 🎸 Tradutor Rítmico

Converte batidas de violão em figuras musicais da partitura. Interface visual, responsiva e mobile-first.

## ✨ Features

- **Builder visual** — monta cada beat tocando botões ↓ ↑ ✋
- **15 padrões preset** — incluindo Samba, Bossa Nova, Baião, Forró, Valsa, Metal e mais
- **Modo manual** — digita direto no formato `v|v^|-^|v`
- **BPM em tempo real** — slider com atalhos rápidos (40 / 60 / 80 / 100 / 120 / 160)
- **Teclado de acordes** — notas A–G + modificadores (m, 7, M7, sus2, sus4, #, ♭)
- **Inferência de compasso** — detecta automaticamente 2/4, 3/4, 4/4, 6/8...
- **Tabela de figuras** — referência sempre visível (Semibreve até Fusa)
- **Dark / Light mode**

## 🚀 Como usar

Abra o arquivo `ritmo-master.html` diretamente no navegador — sem instalação, sem dependências.

Ou acesse via [GitHub Pages](https://victoredbr21.github.io/tradutor-ritmico/ritmo-master.html) após ativar nas configurações do repositório.

## 🎵 Legenda dos strums

| Símbolo | Ação |
|---------|------|
| ↓ | Palhetada para baixo |
| ↑ | Palhetada para cima |
| ✋ | Pausa / mute |
| `|` | Divisor de tempo (beat) |
